# 荧光笔

荧光笔(支持当代码块没有语言标签时在代码块中使用)

  • <h-lighter color="COLOR">WORD</h-lighter>
  • WORD替换为要高亮的文字
  • color属性可选,COLOR为高亮颜色,不填color默认为黄色
  • COLOR支持css颜色关键词/RGB/16进制颜色

这是一个<h-lighter>高亮</h-lighter>示例,你可以<h-lighter color="pink">切换</h-lighter><h-lighter color="rgb(33, 168, 252,0.6)">高亮</h-lighter>的<h-lighter color="#dafbe1">颜色</h-lighter>

inline int read() {
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-')
            f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + ch - '0';
        ch = getchar();
    }
    return x * f;
}
inline void write(int x)
{
    if (x < 0) { putchar('-'); x = -x; }
    if (x > 9) write(x / 10);

}
