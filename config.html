# 配置

# 主题配置

# 顶部导航栏

你可以通过 themeConfig.nav 增加一些导航栏链接


例子

// .vuepress/config.js
module.exports = {
  themeConfig: {
    nav: [
      { text: '展示', link: '' },
      { text: '关于', link: 'about' },
      { text: '文档', link: 'doc' },
      { text: '问题', link: 'question' },
      { text: '进一步了解', link: 'more' },
      { text: '更新日志', link: 'log' }
    ],
  }
}

# 侧边章节栏

你可以通过 themeConfig.group 增加侧边栏中的章节栏部分

# 基本配置

在下面的例子中'doc':[]意味着只有当前链接在base/doc之下侧边栏才会有下面章节栏

group:{
  'doc':[
    { text:'文档',link: 'doc' },
    { text: '快速开始', link: 'doc/quick-start' },
    { text: '配置', link: 'doc/config' },
    { text: '插件', link: 'doc/plugin' },
    { text: 'Markdown', link: 'doc/markdown' },
    { text: '其他', link: 'doc/other' }
  ]
},

# 子章节栏

你可以将link换为sublink来设置子侧边栏


例子
如你可以将:

{ text: '插件', link: 'doc/plugin' }

换为

{
 text: '插件',
 sublink: [
   {
     text: '插件目录',
     link: 'doc/plugin'
   },
   {
     text: '图片增强插件',
     link: 'doc/plugin/el-img'
   }
 ]
 }

# 章节栏分组

分组与子章节栏的区别

两者不同在于分组并没有实际上创建一个下拉框,并且分组可以嵌套子章节栏(见示例二),但反之不行

你可以将原本的link换位subgroup来分组,text变为组名

{
  text: '重要部分',
  subgroup: [
    { text: '快速开始', link: 'doc/quick-start' },
    { text: '配置', link: 'doc/config' },
    { text: 'Markdown', link: 'doc/markdown' }
  ]
},

# 代码高亮相关配置

你可以通过

  • themeConfig.HighlightOptions来配置代码高亮主题选项,
  • themeConfig.defaultHighlight来配置默认应用的代码高亮主题
  • themeConfig.limitHightlight来配置最多同时应用的代码高亮主题数量
  • 代码高亮主题名请与highlight.js@11.8.0一致
// 下面是默认配置
themeConfig: {
    HighlightOptions:[
        "atom-one-light",
        "atom-one-dark",
        "github",
        "github-dark",
        "monokai",
        "tokyo-night-dark",
        "tokyo-night-light",
        "pojoaque",
        "sunburst",
        "vs",
        "panda-syntax-dark",
        "panda-syntax-light",
        "googlecode",
        "night-owl",
        "mono-blue",
        "school-book",
        "stackoverflow-dark",
        "stackoverflow-light",
        "tomorrow-night-blue",
        "tomorrow-night-bright",
        "rainbow",
      ],
    defaultHighlight:["atom-one-light", "atom-one-dark"],
    limitHightlight:5
}

# 编辑

你可以通过themeConfig.editLinkText来配置编辑按钮的链接(注意编辑按钮是基于git的)
例如本网站:editurl: 'https://github.com/open17/vuepress-theme-easy-book/blob/master/example'

# 加密文章

并不是严格意义上的加密,依然可以在源代码中找到密码

# 设置密码

themeConfig:{
  lock_password: '123456',
}

# 加密

你可以在文章中使用lock来加密文章

---
lock: true
---

# 最大搜索

搜索栏给出的最大搜索建议数量，默认为50,你可以通过在配置文件中添加以下内容来更改此设置：

themeConfig:{
  searchMaxSuggestions:25,
}
