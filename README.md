## 功能
- v1.2.x重构UI,支持封面
- 支持在线切换多种代码高亮(atom-one,github,monokai,tokyo-night等)
- 本地全文内容搜索
- 移动端网页端适配
- 支持明暗双模式
- Markdown增强:视频插入,选项卡,图片增强,荧光笔,代码复制...
- 工具栏: 一键编辑,回到顶部,章节,工具栏折叠...
- 其他功能: h1-h6目录,章节分组,多文件目录,自定义导航栏...
- 持续维护中

## 快速开始
### 安装
```shell
npm i vuepress-theme-easy-book -D
```
### 配置
```js
// .vuepress/config.js
module.exports = {
    //使用本主题
    theme: 'easy-book',
}
```
